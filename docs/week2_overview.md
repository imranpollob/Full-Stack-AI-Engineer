# Week 2: Data Science Essentials - The Math Behind the Magic

Welcome to Week 2 of the Full-Stack AI Engineer bootcamp! Now that we have mastered Python's fundamentals in Week 1, we are ready to tackle the real data tools.

Artificial Intelligence is essentially advanced mathematics applied to massive datasets. If you try to do this math using standard Python loops and lists, your code will be impossibly slow. This week is all about high-performance data manipulation and visualization.

## What We'll Cover This Week

This week, we will introduce the "Holy Trinity" of Python Data Science libraries: **NumPy**, **Pandas**, and **Matplotlib/Seaborn**.

Here is our daily breakdown:
*   **Day 1: Introduction to NumPy.** We'll learn how to use NumPy arrays. These highly optimized C-backed data structures allow us to perform matrix math at blistering speeds.
*   **Day 2: Advanced NumPy.** We dive into Broadcasting, boolean indexing, random number generation, and executing mathematical aggregations across multiple dimensions.
*   **Day 3: Introduction to Pandas.** Pandas is the undisputed king of tabular data. We'll learn to load CSVs and Excel files into "DataFrames" for analysis.
*   **Day 4: Data Cleaning.** Real-world data is disastrous. We'll learn how to handle missing values (NaNs), transform columns, and merge multiple massive datasets together.
*   **Day 5: Aggregation and Grouping.** Learn how to slice your data by categories, compute statistics, and find high-level trends using `groupby` and pivot tables.
*   **Day 6: Data Visualization.** Numbers on a screen are hard to read. We will use Matplotlib and Seaborn to generate beautiful line charts, scatter plots, and heatmaps to reveal the secrets hidden in our data.
*   **Day 7: The EDA Project.** We wrap the week up by putting everything into practice: Exploratory Data Analysis (EDA) on a real dataset.

## Why This Matters for AI
You cannot train an AI model on bad data. The phrase "Garbage In, Garbage Out" is the golden rule of Machine Learning. The skills you learn this week—cleaning, aggregating, organizing, and visualizing data—will make up 80% of your actual day-to-day job as an AI Engineer.
