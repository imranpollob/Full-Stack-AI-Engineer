# Week 3: Mathematics for Machine Learning

Welcome to Week 3 of the Full-Stack AI Engineer bootcamp! We've mastered the building blocks (Python) and the data wrangling tools (NumPy, Pandas, Matplotlib). Now, we reach the literal core of Artificial Intelligence: Mathematics.

Machine learning is not magic. It is essentially calculus and probability applied to massive matrices of data. If you truly want to be a full-stack AI engineer rather than just someone who copies code from the internet, you must understand the underlying math.

## What We'll Cover This Week

This week serves as a crash course in the three pillars of AI mathematics: **Linear Algebra**, **Calculus**, and **Statistics/Probability**. We will write actual Python code that proves the math concepts.

Here is the breakdown of our daily tutorials:
*   **Day 1: Linear Algebra Fundamentals.** We'll define Vectors, Matrices, and perform fundamental operations (addition, scalar multiplication, and the crucial dot product matrix multiplication).
*   **Day 2: Advanced Linear Algebra Concepts.** We dive deeper into matrix properties (Determinants and Inverses), Matrix Decompositions (SVD), and Eigenvalues—the math powers dimensionality reduction algorithms like PCA.
*   **Day 3: Calculus - Derivatives.** The engine of neural network learning. We'll learn how to calculate partial derivatives, find mathematical gradients, and introduce Gradient Descent.
*   **Day 4: Calculus - Integrals and Optimization.** We'll explore integrals for continuous probabilities and tackle advanced optimization variations like Stochastic Gradient Descent (SGD) and Adam.
*   **Day 5: Probability Theory and Distributions.** AI models operate on uncertainty. We'll cover Bayes' Theorem, Gaussian distributions, and other probability models that power both classification and NLP tasks.
*   **Day 6: Statistics Fundamentals.** We'll solidify core concepts like Variance, Standard Deviation, Hypothesis Testing, and P-Values, which are necessary to prove if your model is actually working or just getting lucky.
*   **Day 7: The Mini-Project - Linear Regression from Scratch!** We combine all three branches of math into a single project. We won't use Scikit-Learn; we will build and optimize a predictive regression model literally from scratch using gradient descent.

## Why This Matters for AI
When a neural network "learns", what is actually happening? It is calculating the calculus gradient (Day 3) of a loss function and updating its matrix weights (Day 1) to maximize the probability (Day 5) of the correct prediction. 

The libraries do the heavy lifting for you, but when an AI model stops learning and you don't understand the math, you won't be able to fix it.
